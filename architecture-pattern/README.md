## 架构视图

### 功能视图

  * B端复杂系统用例图
  * C端产品交互设计稿或UI原型图
  

### 逻辑视图

  * 逻辑模块划分（数据结构、面向对象设计里的类图、状态图）

### 物理视图
   
  * 整个系统所在机房、各类机器数目、机器配置、网络带宽
   
### 开发视图

  * 工程结构、目录结构、jar包

### 运行视图

  * 系统多进程、多线程之间的同步
