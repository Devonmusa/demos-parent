##  项目结构

    demos-parent
        |
        |------------doc  开发者必备技能文档
        |
        |
        |------------algorithm-demo 数据结构与算法基础
        |
        |
        |------------distrebuted-demo 分布式解决方案:服务发现、分布式缓存、负载均衡

        |
        |------------java-demo  java原理文档及相应代码实现基础demo，包括java基础、设计模式、并发、IO、JVM参数配置
